# Netflix Exploratory Data Analysis (EDA)

## Objective

The objective of this project is to perform Exploratory Data Analysis (EDA) on the Netflix dataset to identify trends, patterns, and insights related to content distribution, release trends, ratings, and genres.

## Dataset

Dataset: Netflix Dataset

File Used: netflix_titles.csv

Number of Rows: 8807

Number of Columns: 12

## Tools and Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* VS Code
* Jupyter Notebook

## Steps Performed

### 1. Data Loading

The Netflix dataset was loaded using Pandas.

### 2. Data Inspection

Dataset structure, shape, data types, and missing values were examined.

### 3. Summary Statistics

Statistical summaries and dataset characteristics were analyzed.

### 4. Content Distribution Analysis

Movies and TV Shows were compared to understand content distribution on Netflix.

### 5. Rating Analysis

Content ratings were analyzed to identify the most common audience categories.

### 6. Trend Identification

Release year trends were studied to understand content growth over time.

### 7. Country Analysis

Top content-producing countries were identified.

### 8. Genre Analysis

Popular genres available on Netflix were analyzed.

## Visualizations Created

* Movies vs TV Shows Bar Chart
* Rating Distribution Chart
* Release Trend Graph
* Top Countries Chart
* Top Genres Chart

## Results

* Movies are more common than TV Shows on Netflix.
* Content production increased significantly after 2010.
* The United States contributes the largest amount of content.
* TV-MA is one of the most common ratings.
* International Movies, Dramas, and Comedies are among the most popular genres.
